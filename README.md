# Python Projects

Collection of small Python projects created for experimentation, learning, and practice.

---

## ENG

This repository contains several independent Python projects organized by category.

### 1. Experimentation/

The `Experimentation` folder contains experimental projects.

Main file: `exper.py`

This project is a procedural artificial intelligence simulation running in a virtual environment.  
It can be described as a simplified MMORPG-like world simulation where an AI entity acts autonomously.

The AI is not based on machine learning. It follows scripted logic and makes reasonable decisions depending on the situation and internal conditions.  
The goal of this project was to explore:

- Procedural behavior
- Decision-making systems
- World simulation logic
- Entity state management

---

### 2. Games/

The `Games` folder contains small game projects.

#### Wordle.py

A recreation attempt of the well-known word guessing game Wordle.  
The project focuses on:

- Input validation
- Word comparison algorithms
- Game loop logic

#### mini_doom.py

An experimental project inspired by the original Doom rendering style.  
Graphics are simulated directly in the console using characters to represent walls and depth.

The goal of this project was to explore:

- Basic raycasting concepts
- Console rendering techniques
- Perspective simulation in text mode

---

### 3. Games/CureForMyBoredom/

Main file: `Mini_Games.py`

This project is a small console-based entertainment collection created mainly as a design and probability experiment.

It includes simulations of:

- Roulette
- Slot machines
- Case opening mechanics

The main goal was to experiment with:

- Probability systems
- Randomness balancing
- Console UI design (colors, layout, visual effects)
- Game feedback systems

This project was created as a way to experiment and build small interactive systems while improving code structure and probability logic implementation.

## FR

Fichier principal : `exper.py`

- Gestion d’état d’entités
